# 자동차보수도장기능사 모의시험 웹 패키지

## 포함 기능
- 접속 시 랜덤 60문항
- 카테고리 편중 완화
- 60분 타이머
- 시험 시작 전 닉네임 입력
- 모든 문항 답변 전에는 제출 불가
- 미응답 문항이 있으면 가장 번호가 작은 미응답 문항으로 이동
- 상단 팝업: "모든 문항에 답변을 체크해주세요"
- 제출 후 정답과 해설 공개
- Firebase Firestore 기반 랭킹 저장
- 동일 닉네임 기준 같은 문제 3회 이상 오답 시 해설 위 경고 표시

## 배포 전 해야 할 일
1. firebase-config.js에 실제 Firebase 설정값 입력
2. firestore.rules를 Firestore 규칙에 붙여넣고 게시
3. 저장소 루트에 파일 업로드
4. GitHub Pages 활성화
